## Laravel Setup

A script to scaffold a Laravel project with the following:
- UI
  - Vite
  - Oruga
  - Bulma
  - Eslint
  - Prettier
- Backend
  - Breeze
  - Sanctum
  - Horizon
  - Spatie permissions
  - Spatie tags
  - Wyxos Laravel Resources

## Usage
``` npx github:wyxos/laravel-setup ```
